# counter_app
Counter_app

• Created a React functional component Counter.

• Initialized the counter value using the useState hook.

• App is provided with Increment, Decrement, and Reset buttons.

• Incremented the counter by 1 on click of Increment Button, • Decremented the counter by 1 on click of Decrement Button
  On clicking the Reset button initializes the counter value to 0.

• Used conditional rendering to show messages such as: “Minimum count reached” when count is below zero, otherwise the counter value is returned.
